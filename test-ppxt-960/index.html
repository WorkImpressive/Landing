Основная система фильтрации
Уникальный водяной фильтр AquaBOX c инновационной технологией подавления пыли Wet-Jet
на входе в пылесос создается мощная водяная стена
пыль из входящего воздушного потока пропитывается водой
мокрая и тяжелая пыль отделяется от чистого воздуха
тяжелая пыль оседает на дне аквабокса и остается надежно запертой в нем
обратно в помещение выходит очищенный воздух
для уборки нужен всего 1 л обычной воды
грязная вода просто выливается по окончании уборки – без контакта с сухой пылью
Во время уборки удерживается 99,99% пыли, 100% пыльцы.
Фильтр тонкой очистки НЕРА13 служит для удержания мельчайших частиц пыли. Предназначен для длительного использования. Произведен из HEPA - материала, который можно мыть.
